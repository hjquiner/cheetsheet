What is Semantic HTML?

Sematic HTML is a specific formatting style used when constructing an HTML page to define elements.

Why use Semantic HTML?

The primary goal is to make the code easier to read and condense the amount of necessary code.
Other benefits include:

  • Keywords can be considered by Search Engines
  • Accessability for the visually impared using screen readers
  • Finding meaningful blocks of code
  • Accurately describing types of data
  • Helps to mirror semantic naming conventions in CSS
Semantic Elements
article Defines an independent item of content.
aside A section of content indirectly related to the main content.
details A custom widget where information is only visible in an "open" state.
figcaption An independent caption related to a parent [figure] element.
figure Self contained content with optional captioning.
footer Contains information pertaining to document origin information.
header Contains introductory content and navigational aids.
main Represents the main body of a document.
mark Highlights specific text elements to convey importance.
nav section to provide navigational links.
section Represents a standalone section of the document.
summary Specifies the text inside a [details] element.
time Formats time-related elements.
